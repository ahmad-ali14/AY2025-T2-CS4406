Guide

  • All demos have the same layout, with the following elements (see image below):
    • Sidebar: Right sidebar starting with the title of the unit.
    • Global options: Controls camera, light, and renderer settings; it is available in all demos.
    • Unit-specific options: Controls specific settings for the current unit.
    • Help button: Opens a modal with information about the demo.
    • Scene: The main area where the 3D scene is displayed.
